# trade-decision

FSI-native, Alpha-Vantage-first trade decision system for Claude Code. Status: Phase 1 (data engine) — under construction.

## Requirements
- Claude Code ≥ 2.x, Python ≥ 3.10 (stdlib only — no pip installs)
- Alpha Vantage API key exported as `ALPHAVANTAGE_API_KEY` (premium 75 req/min recommended; free tier degrades with disclosure)
